Q: Explain CSS Flexbox vs Grid?

A: Flexbox is one-dimensional (row/column), Grid is two-dimensional (rows & columns). Use Flex for alignment; Grid for full layouts.

Q: What are React hooks?

A: Functions like useState/useEffect for state & side-effects in functional components.

Q: Debouncing vs Throttling?

A: Debounce waits for pause; Throttle limits calls in an interval.

Q: HTTP vs HTTPS?

A: HTTPS encrypts data via TLS/SSL, ensuring confidentiality & integrity.
